### 二、const 修饰指针变量。

const 修饰指针变量有以下三种情况。

* A: const 修饰指针指向的内容，则内容为不可变量。
* B: const 修饰指针，则指针为不可变量。
* C: const 修饰指针和指针指向的内容，则指针和指针指向的内容都为不可变量。

对于 A:

```
const int *p = 8;
```

则指针指向的内容 8 不可改变。简称左定值，因为 const 位于 * 号的左边。

对于 B:

```
int a = 8;
int* const p = &a;
*p = 9; // 正确
int  b = 7;
p = &b; // 错误
```
